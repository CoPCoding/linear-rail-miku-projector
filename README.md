# Linear Rail Miku Projector
A linear rail assembly to simulate a hologram effect of a virtual avatar moving across a range and save on space

Features:
* Optimizes the area the hologram can cover by simply capturing just the character whenever they're present.
* More realisitc hologram presentation by not having a whole sheet of rear projection film and somewhat breaking the illusion.
* Auto-calibration and adjustable constants for varying scales.
